FMS Drills

Welcome to Fine Motor Skills Training.

This application contains 3 exercises that will help tune fine motor skills for patients who have suffered a stroke. Each exercise provides feedback to let you know how you are doing, and certain exercises are randomized every time you begin the program, to provide an extra challenge, and some versatility. We hope you find this useful.

Let's get started!

Maze Game

Type of Exercise: Follow pattern
Description: This exercise is a maze with a dot at the starting point, the user will use the stylus to get the dot to the end of the maze. The goal is to not touch the walls of the maze while tracing through. Mazes are randomly generated each playthrough.
Screen: Shows the instructions and maze.
User: Uses stylus to drag dot through the maze.
System Provides: Positive feedback if the user completes the maze. Negative feedback if the user touches a wall.
