Rhythm Game

Type of Exercise: Repetetive movement
Description: The user will see a circle go across the screen and get into a rhythm of clicking the mouse when the circle reaches a certain point. Practices fine motor skills using a mouse. The exercise will cover the pain point of hand dominance, where the user will repetitively click the mouse to the sight of the circle.
Screen: Shows the instructions and the game.
User: Uses one index finger to click the mouse in time.
System Provides: Positive feedback if the user is in time enough. Negative feedback if the user is out of time.
