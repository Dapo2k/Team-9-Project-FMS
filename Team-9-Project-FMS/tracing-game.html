Tracing Game

Type of Exercise: Drawing
Description: The user will be provided with a randomly generated line and will need to trace that line to practice FMS. The results will be the time it takes to trace the line.
Screen: Shows a randomly generated line.
User: Uses stylus to trace the randomly generated line.
System Provides: The line will turn green once the user has traced it correctly and will be red if the user hasn't traced it yet. The yellow point will indicate the start of the line.
